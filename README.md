# 📊 Task 1: Sales Performance Dashboard (Excel)

This project was completed during my internship at **Elevvo Pathways**. It involves the creation of a comprehensive Sales Performance Dashboard using the "Superstore" dataset to track business health and regional performance.

---

## 🎯 Project Objective
The goal was to transform raw transactional data into an interactive visual tool that allows stakeholders to monitor revenue trends, product performance, and regional sales distribution at a glance.

## 🛠️ Skills & Techniques Applied
* **Data Cleaning:** Handling missing values and formatting the Superstore dataset for analysis.
* **Pivot Tables:** Summarizing thousands of rows of data into actionable business metrics (Revenue, Units Sold, Profit).
* **Advanced Visualization:** Creating dynamic charts that update automatically.
* **Interactivity:** Implementing **Slicers** and **Timelines** to allow users to filter data by Region, Category, and Time Period.

---

## 📈 Key Metrics Tracked
* **Revenue Trends:** Monthly and quarterly sales growth analysis.
* **Product Performance:** Top-performing categories and sub-categories by units sold.
* **Regional Insights:** A breakdown of sales across different geographic territories.
* **Profitability:** Identifying which segments drive the highest margins.

---

## 🚀 How to View the Project
1. Download the `.xlsx` file from this repository.
2. Open it in **Microsoft Excel**.
3. Use the **Slicers** on the dashboard tab to interact with the data and see real-time updates across all charts.
